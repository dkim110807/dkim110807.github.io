## 문제
[BOJ 1000 - A+B](https://acmicpc.net/problem/1000)

## 풀이
두 수를 입력받아 합을 출력하면 된다.

## 코드
```cpp
#include <bits/stdc++.h>

int main() {
  int a, b;
  std::cin >> a >> b;
  std::cout << a + b;
}
```
